### Problem
The Telegram bot wasn't responding to /start commands because:
1. The Telegram bot token wasn't configured in `.env` (was set to placeholder `your-telegram-bot-token`)
2. The Telegram integration router wasn't included in `main.py`

### Solution
1. Updated `.env` with the correct Telegram bot token:
   ```
   TELEGRAM_BOT_TOKEN=8504942821:AAHkNDF5Fbs2RNPiAwYjXNONoKXHVGKsk8Y
   TELEGRAM_WEBHOOK_URL=https://your-domain.com/integrations/telegram/webhook
   ```

2. Added Telegram router to `main.py`:
   ```python
   from app.integrations.telegram import router as telegram_router
   # ...
   app.include_router(telegram_router)
   ```

3. Rebuilt and restarted Docker containers

4. Set up webhook via Telegram API:
   ```
   POST https://api.telegram.org/bot<token>/setWebhook?url=https://desiccated-superoratorical-denisha.ngrok-free.dev/integrations/telegram/webhook
   ```
